# 智能水槽三维实时监控系统

这是一个基于 **Vue 3 + Vite + Three.js** 的智能水槽三维实时监控前端项目，用于展示水槽工况、闸门状态、传感器数据、实时趋势、历史曲线和报警处理等内容。

项目界面采用深色科技风与工业监控大屏风格，适合智慧水利实验、明满流混合实验、水槽运行监测和设备联动控制场景。

## 功能概览

- 三维水槽工艺图展示
- 闸门实时状态监控
- 传感器实时状态监控
- 手动控制闸门和水泵
- 实时趋势分析图
- 历史数据查询与回放
- 报警列表与报警处理弹窗

## 技术栈

- Vue 3
- Vite
- Three.js

## 本地运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

默认访问地址：`http://127.0.0.1:4173/`

### 3. 构建生产版本

```bash
npm run build
```

### 4. 本地预览构建结果

```bash
npm run preview
```

## 资源生成

如果需要重新生成三维水槽模型，可以运行：

```bash
npm run generate:flume-model
```

## 项目结构

- `src/`：Vue 应用源码
- `scripts/`：模型生成脚本
- `models/`：三维模型和预览文件
- `output/`：输出结果与验证截图
- `智能水槽系统设计图与说明/`：页面原型和设计说明
