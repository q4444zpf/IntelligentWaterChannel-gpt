# 智能水槽系统实时监控首页原型布局说明

## 1. 页面定位

本页面用于展示智能水槽系统的实时运行状态，面向智慧水利实验平台、明满流混合实验水槽、闸门控制实验、水泵调度实验等场景。

页面核心目标是通过三维数字孪生工艺图，将水槽中的闸门、水泵、渠道、倒虹吸、传感器、报警和实时趋势数据统一展示，并支持设备控制和异常联动。

---

## 2. 页面整体风格

页面采用 **深色科技风 / 数字孪生大屏风格 / 工业监控平台风格**。

整体要求：

- 16:9 横屏布局；
- 深蓝、黑蓝、荧光蓝为主色；
- 绿色表示在线 / 正常；
- 黄色表示预警；
- 红色表示异常 / 报警；
- 灰色表示离线；
- 页面具有高保真产品 UI 质感；
- 中间三维水槽工艺图为视觉中心；
- 左侧偏状态总览；
- 中间偏三维监控与趋势分析；
- 右侧偏设备控制与报警信息。

---

## 3. 整体布局

页面采用 **三列布局**。

```text
┌────────────────────────────────────────────────────────────────────────────┐
│ 顶部状态栏                                                                 │
├───────────────┬──────────────────────────────────────┬─────────────────────┤
│ 左列            │ 中列                                   │ 右列                │
│               │                                      │                     │
│ 左上：闸门实时状态 │ 中上：三维水槽工艺图                    │ 手动控制设备          │
│               │                                      │                     │
│ 左下：传感器状态 │ 中下：实时趋势分析图                    │ 实时报警信息          │
└───────────────┴──────────────────────────────────────┴─────────────────────┘
```

建议宽度占比：

| 区域 | 宽度占比 | 说明 |
|---|---:|---|
| 左列 | 20% | 闸门状态、传感器状态 |
| 中列 | 55% | 三维水槽工艺图、实时趋势分析 |
| 右列 | 25% | 手动控制、报警信息 |

高度划分建议：

| 区域 | 高度占比 | 说明 |
|---|---:|---|
| 顶部状态栏 | 8% | 系统标题、工况、连接状态 |
| 主体内容区 | 92% | 三列主体区域 |

中列内部高度建议：

| 区域 | 高度占比 |
|---|---:|
| 中上三维水槽工艺图 | 58% |
